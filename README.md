This repository contains code, assignments, and auxiliary files from my disciplines in the Mathematics program at the [Instituto de Ciências Matemáticas e de Computação (ICMC)](https://pt.wikipedia.org/wiki/Instituto_de_Ci%C3%AAncias_Matem%C3%A1ticas_e_de_Computa%C3%A7%C3%A3o_da_Universidade_de_S%C3%A3o_Paulo), [University of São Paulo (USP)](https://en.wikipedia.org/wiki/University_of_S%C3%A3o_Paulo).

## Structure

* `SME0XXX_Name_of_Programming_Discipline/` 
    * `/src`: Source code for programming assignments and projects.
    * `/docs`: Reports, notes, and auxiliary files.
* `SME0YYY_Name_of_Theoretical_Discipline/`
    * `/lists`: PDF files of exercise lists provided by the professor.
    * `/solutions`: PDFs of completed exercises and notes.
* `general_files/`: Shared scripts, templates, and other general academic files.
